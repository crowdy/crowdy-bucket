# crowdy-bucket

[Scoop](https://scoop.sh/) bucket for crowdy tools.

## Usage

```powershell
scoop bucket add crowdy https://github.com/crowdy/crowdy-bucket
scoop install conoha
```

## Available Apps

| App | Description |
|-----|-------------|
| [conoha](https://github.com/crowdy/conoha-cli) | CLI tool for ConoHa VPS3 API |

Manifests are automatically updated by [GoReleaser](https://goreleaser.com/) on each release.
